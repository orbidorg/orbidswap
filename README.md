# OrbIdSwap

DeFi Protocol on World Chain.

## Structure

This is a monorepo containing:

- **`/contracts`**: Smart contracts (Uniswap V2 Fork) and Hardhat scripts.
- **`/frontend`**: Next.js application.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Contracts**:
    ```bash
    npm run contracts:compile
    npm run contracts:test
    ```

3.  **Frontend**:
    ```bash
    npm run frontend:dev
    ```
